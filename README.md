# 吉他開放把位和弦查詢網站

這是一個單頁網站，開啟 `index.html` 就可以使用。

## 放到 GitHub Pages 的基本流程

1. 到 GitHub 建立一個新的 repository。
2. 把 `index.html` 上傳到 repository 的根目錄。
3. 進入 repository 的 `Settings`。
4. 找到 `Pages`，選擇從 `main` 分支的根目錄發布。
5. 儲存後等待 GitHub 產生公開網址。

如果之後要修改網站，只要更新 `index.html` 後重新上傳或 commit 即可。

官方說明：https://docs.github.com/en/pages/getting-started-with-github-pages/configuring-a-publishing-source-for-your-github-pages-site
